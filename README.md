# Mohamed Arsath — Glassmorphism Portfolio

Static single-page portfolio built with HTML/CSS/JS featuring a light gradient background and glass UI.

## Quick Start

1. Open `index.html` in your browser (double-click) to preview locally.
2. Place your profile photo at `assets/profile.jpg` (512×512 JPG recommended).

## Edit Contact

- The contact form uses `mailto:`. Replace `your.email@example.com` in `index.html` with your email.
- Optional: Switch to Formspree by replacing the form `action` with your Formspree endpoint.

## Deploy to GitHub Pages

1. Create a repository and push this folder:
   ```bash
   git init
   git add .
   git commit -m "feat: glass portfolio"
   git branch -M main
   git remote add origin https://github.com/<your-username>/<repo>.git
   git push -u origin main
   ```
2. In GitHub, open Settings → Pages → Build and deployment:
   - Source: Deploy from a branch
   - Branch: `main` / root
3. Your site will be available at `https://<your-username>.github.io/<repo>/`.
